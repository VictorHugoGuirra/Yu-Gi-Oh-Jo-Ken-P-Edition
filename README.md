# 🎴 Yu-Gi-Oh Jo-Ken-Pô Edition  

Um mini-game interativo que une o clássico **Pedra, Papel e Tesoura** ao universo de **Yu-Gi-Oh!**.  
Escolha sua carta, desafie o computador e veja quem vence no duelo! 🃏⚡  

---

## ✨ Funcionalidades  

- 🔊 **Trilha sonora infinita** que inicia automaticamente.  
- 🎥 **Background animado** com vídeo em loop.  
- 🃏 **Sistema de cartas** baseado em Pedra, Papel e Tesoura.  
- 🏆 **Placar dinâmico** (vitórias e derrotas).  
- 💫 **Efeitos visuais e cursores personalizados** para mergulhar no clima do jogo.  
- 🎮 Feito totalmente em **HTML, CSS e JavaScript puro**.  

---

## 🚀 Como jogar  

1. Abra o jogo no navegador.  
2. Passe o mouse sobre as cartas para visualizar detalhes.  
3. Clique em uma carta para desafiar o computador.  
4. O placar será atualizado automaticamente com cada rodada.  

---

# Estrutura do Projeto

- **index.html**: Estrutura principal da página.

- **src/**
  - **assets/**: Contém todos os arquivos de mídia e elementos visuais.
    - **audios/**: Música de fundo do jogo.
    - **cursor/**: Cursores personalizados.
    - **icons/**: Cartas e ícones do jogo.
    - **rpg/**: Botões, bordas e fundos temáticos.
    - **video/**: Background animado do jogo.
  - **scripts/**: Contém a lógica do jogo.
    - **engine.js**: Script principal do jogo.
  - **styles/**: Arquivos CSS para estilização.
    - **buttons.css**: Estilização dos botões.
    - **containers_and_frames.css**: Estilização de containers e molduras.
    - **main.css**: Estilos principais gerais.
    - **reset.css**: Reset de estilos padrões do navegador.

---

## 🛠️ Tecnologias utilizadas  

- **HTML5** → Estrutura da aplicação.  
- **CSS3** → Estilização, cursores customizados e elementos visuais.  
- **JavaScript (ES6)** → Mecânica do jogo e placar dinâmico.
